ACCORDION (AULA 16)


ACCORDION -> São úteis quando você deseja alternar entre ocultar e mostrar grande quantidade de conteúdo:

O acordeão ("ACCORDION") Bootstrap é um componente que organiza o conteúdo dentro de itens recolhíveis. O acordeão permite a exibição de apenas um item recolhido por vez. Os acordeões podem alternar entre vários blocos de texto com um único clique e isso aprimora muito a experiência do usuário do seu projeto.

FORMULÁRIOS (AULA 18)


FORMULÁRIOS->Exemplos, instruções para estilizar campos de formulários, opções de layout e componentes personalizados para criar uma grande variedade de formulários.
Campos de formulário textuais como < input>, < select> e < textarea> são estilizados com a classe .form-control. Ela possui estilos para aparência, estado de foco e muito mais.
* Para aumentar o campo em altura é so usar o (class="form-control form-control-lg"). / Para diminuir é so usar o "SM" no ligar o "LG"
* Uma propriedade extra você pode usar o "READONLY"{PROPRIEDADE PARA QUE O SEU USUARIO NÃO ALTERE O TEEXTO DENTRO DO CAMPO}
* FORM-CONTROL-PLAINTEXT - interessante.
** PARA COLOCAR O SEU FORMULÁRIO INLINE (EM LINHA / NA HORIZONTAL).
** EXPLICAÇÃO MUITO IMPORTANTE PARA SE USAR OS GRIDS ( FORM-ROW) ***** MUITO INTERESSANTE.******

INPUT GROUP(AULA 19)


INPUT GROUP->"Grupo de entrada"

Amplie facilmente os controles de formulário adicionando texto, botões ou grupos de botões em ambos os lados das entradas de texto, seleções personalizadas e entradas de arquivo personalizadas.

A input-group é um contêiner para aprimorar uma entrada adicionando um ícone, texto ou um botão na frente ou atrás do campo de entrada como um "texto de ajuda".
